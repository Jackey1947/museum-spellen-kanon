# Museum Kanon — Belegering van Maastricht

Een interactief browserspel voor museumbezoekers over de belegering van Maastricht. Spelers richten een kanon op de walmuur en ontdekken bij welke hoek een bres ontstaat.

**Speel het spel:** https://jackey1947.github.io/museum-spellen-kanon/

## Hoe het spel werkt

1. Gebruik de **−** en **+** knoppen om het kanon te richten (10° – 55°).
2. Druk op **VUUR** om een kanonschot af te vuren.
3. Bij de juiste hoek (30°) raakt de kogel de walmuur en ontstaat er een bres — met geluid en puin-animatie.
4. Bij een andere hoek mist de kogel en reset het spel automatisch.
